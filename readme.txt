Staňte sa partnerom Martinus.sk a propagujte na svojich web stránkach knihy ktorá sa vám páčia.

== Description ==

Martinus partnerský systém umožňuje pridať a zobraziť na stránke banner z partnerského systému kníhkupectva Martinus.sk:

 * 3 veľkosti bannerov.
 * Banner na konkrétnu knihu
 

== Installation ==

Môžete nainštalovať pludin priamo vo Vašom WP:

 1. v menu *Moduly* kliknite na *Pridať nový*.
 2. Vyhľadajte text *Martinus*.
 3. Kliknite na *Nainštalovať teraz*.
 4. Aktivujte modul.

= Nastavenie =

1. Menu Nastavenia -> Martinus PP nastavte Váš partnerský kód.
2. Menu Vzhľad -> Widgety vyhľadajte widget *Martinus partnerský systém*.
3. vložte widget do vybranej oblasti widgetov
4. nastavte typ baneru a pridajte linku na knihu z eshopu www.martinus.sk (napr. http://www.martinus.sk/?uItem=54475)
5. Uložte, banner sa začne zobrazovať automaticky.

Banner je tiež možné vložiť do tela článku alebo stránky.
